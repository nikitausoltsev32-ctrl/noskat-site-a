# ТЗ для Claude: Сайт А — NOSECUT / NOSKAT

## 1. Контекст

Нужно разработать **первый сайт проекта** — информационный портал про **ноускаты / nose cut / передние части автомобилей из Китии**.

Это **Сайт А**.  
Он не является лидогенерационным лендингом и не должен продавать услугу напрямую.

Главная цель сайта — SEO-медиа портал:

- новости китайского авторынка;
- обзоры ноускатов и автомобильных брендов;
- статьи про поставки, логистику, таможню и восстановление авто;
- брендовые SEO-хабы;
- информационные и юридические страницы.

---

## 2. Главный принцип

Не делать сайт как “оставьте заявку / закажите ноускат”.

Сайт А должен быть:

- информационным;
- экспертным;
- SEO-ориентированным;
- визуально сильным;
- похожим на авто-медиа / отраслевой портал.

Лидогенерация, формы подбора, заявки и коммерческие блоки относятся ко второму сайту и здесь не нужны.

---

## 3. Что НЕ добавлять

Не добавлять:

- форму заявки;
- “оставьте заявку”;
- “получить консультацию”;
- “заказать за 25 дней”;
- подбор ноуската;
- калькулятор стоимости;
- коммерческий лендинг;
- прямую продажу услуги.

Допустимы только нейтральные информационные CTA:

- “Читать новости”
- “Смотреть обзоры”
- “Перейти к брендам”
- “Читать статью”

---

## 4. Визуальный стиль

Сохраняем стиль **NOSECUT CHINA / NOSKAT**:

- чёрный / графитовый фон;
- красные акценты;
- белый и серый текст;
- крупные индустриальные заголовки;
- авто-тематика;
- Китай;
- логистика;
- порт / контейнеры;
- карта Китая;
- ноускаты / передняя часть автомобиля;
- ощущение технического автомобильного медиа.

Не делать сайт:

- слишком гламурным;
- слишком корпоративным;
- светлым;
- “синим IT-сайтом”;
- похожим на обычный шаблонный новостник.

---

## 9. Hero

Hero — главный вход в информационный портал.  
Не рекламный блок, не форма заявки и не обычный новостной слайдер.

### Контент Hero

Метка:

```txt
Информационный портал
```

H1:

```txt
Ноускаты из Китая: новости, обзоры, поставки и рынок
```

Подзаголовок:

```txt
Следим за китайским авторынком, поставками ноускатов, логистикой, таможней, брендами и восстановлением автомобилей.
```

Кнопки:

```txt
Читать новости
Смотреть обзоры
```

Chips:

```txt
Новости рынка
Обзоры авто
Поставки
Таможня
Бренды Китая
```

### Визуал Hero

Использовать:

- автомобиль / переднюю часть автомобиля;
- карту Китая;
- красное свечение;
- порт / контейнеры;
- техническую сетку;
- тёмный фон;
- индустриальные линии / overlay.

Новостной слайдер можно оставить ниже Hero как блок **“Главное сегодня”**.

## 19. Cookie banner

Добавить cookie banner:

- “Принять всё”
- “Только необходимые”
- “Настройки”

В настройках:

- необходимые cookie — всегда включены;
- аналитические cookie — toggle;
- кнопка “Сохранить настройки”.

Если делается код:

- сохранять выбор в localStorage;
- не показывать баннер повторно после выбора;
- Яндекс.Метрика подключается только после согласия на аналитику.

## 21. Адаптив

Обязательно:

- desktop 1440px;
- desktop 1200px;
- tablet 768px;
- mobile 390px / 360px.

На mobile:

- карточки в одну колонку;
- hero читаемый;
- chips не ломаются;
- меню компактное;
- sidebar уходит вниз;
- нет горизонтального скролла;
- кнопки не вылезают за экран.
