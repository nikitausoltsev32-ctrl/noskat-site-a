ТЕХНИЧЕСКОЕ ЗАДАНИЕ
на разработку двух сайтов
Тематика: ноускаты из Китая
Формат: 2 отдельных сайта на единой техбазе
1. Общая задача
Нужно разработать два сайта: информационный портал и сайт-заявку. Оба сайта строятся по единой логике: хаб -> карточка -> внутренняя страница. Дизайн с нуля не требуется: допускается сборка из готовых компонентов, но без прямого копирования референсов.
2. Сайт А — информационный портал
Назначение: SEO-трафик, новости, статьи, экспертный и справочный контент.
Тематика: ноускаты из Китая, китайский автомобильный рынок, поставки и логистика, таможня и импорт, обзоры автомобилей, новости брендов, выставки и события, аналитика.
Нужны типовые шаблоны страниц, а не отдельные страницы под каждый материал.
Основные шаблоны:
• Главная
• Список новостей
• Категория новостей
• Список статей
• Категория статей
• Детальная новость / статья
• Страница бренда / тематический хаб
• Универсальная текстовая страница
• Универсальная юридическая страница
• 404
Структура главной страницы:
• Header с меню
• Hero-блок
• Лента последних новостей
• Блок статей / обзоров
• Блок категорий
• Блок популярных материалов
• Блок с брендами / тематиками
• FAQ
• Footer
Страница списка материалов и хабов:
• заголовок раздела;
• короткое описание;
• хлебные крошки;
• фильтр по теме / категории при необходимости;
• карточки материалов;
• нумерованная пагинация;
• блок популярных материалов или связанных разделов;
• footer.
Страница внутреннего материала:
• заголовок;
• короткое вступление;
• основной текст;
• изображения и подзаголовки;
• списки / вставки / таблицы при необходимости;
• блок похожих материалов;
• блок "Поделиться" при необходимости;
• хлебные крошки;
• footer.
Универсальная текстовая страница используется для:
• О проекте
• Контакты
• Реклама
• другие текстовые страницы по необходимости
Универсальная юридическая страница используется для:
• Политика конфиденциальности
• Согласие на обработку персональных данных
• Cookie-политика
• Правила использования cookie
• Пользовательское соглашение
Disclaimer не является обязательным элементом и выносится в отдельный раздел только при необходимости.
3. Сайт Б — сайт-заявка / лидогенерация
Назначение: пользователь оставляет заявку на подбор ноуската, задает вопрос или получает консультацию.
Тематика: подбор ноускатов из Китая, покупка под конкретный автомобиль, консультации по совместимости, доставка и поставка, помощь в подборе деталей.
Нужны типовые шаблоны страниц, чтобы сайт работал как понятная воронка: вход -> доверие -> выбор -> заявка -> подтверждение.
Основные шаблоны:
• Главная / основной лендинг
• Страница заявки
• Страница подбора по марке / модели авто
• Страница преимуществ / почему мы
• FAQ
• Контакты
• Статьи / полезные материалы
• Детальная статья
• Универсальная текстовая страница
• Универсальная юридическая страница
• Страница "Спасибо"
• 404
Главная страница должна содержать:
• Header;
• Hero-блок с главным предложением;
• Короткую форму заявки;
• Блок преимуществ;
• Блок "Как мы работаем";
• Блок подбора по марке / модели;
• Блок доверия / кейсов;
• FAQ;
• Footer.
Страница заявки должна содержать:
• заголовок;
• краткое описание;
• форму заявки;
• поля формы;
• блок с преимуществами;
• блок с тем, что будет после отправки;
• альтернативный способ связи через мессенджеры или телефон;
• footer.
Хаб по автомобилям должен содержать:
• плитку марок автомобилей;
• плитку моделей;
• фильтр или выбор по параметрам;
• карточки автомобилей;
• кнопку перехода к заявке;
• хлебные крошки;
• footer.
Страница преимуществ / почему мы должна содержать:
• список преимуществ;
• блок с этапами работы;
• блок с доверием / кейсами;
• CTA на заявку;
• footer.
FAQ-раздел должен содержать:
• список частых вопросов;
• аккордеон или раскрывающиеся ответы;
• блок с переходом к заявке;
• footer.
Страница контактов должна содержать:
• телефон;
• мессенджеры;
• email;
• адрес;
• карту;
• график работы;
• форму обратной связи;
• footer.
Хаб со статьями / полезными материалами должен содержать:
• плитку или список материалов;
• карточки статей;
• фильтр по темам при необходимости;
• хлебные крошки;
• footer.
Страница "Спасибо" должна содержать:
• сообщение об успешной отправке;
• краткое пояснение, что дальше свяжутся с пользователем;
• кнопку вернуться на главную;
• кнопку связаться через мессенджер;
• блок полезных материалов;
• footer.
4. SEO и навигация
На страницах списков и хабов использовать нумерованную пагинацию вместо бесконечной подгрузки. Предпочтительный формат: 1 2 5 7 8 ... либо аналогичный набор страниц с переходом по номерам. Кнопка "Показать ещё" допустима только как дополнительный вариант, но не как основной.
Для SEO и удобства индексации Яндексом страницы должны переключаться блоками или страницами, а не только длинной непрерывной лентой.
5. Footer и юридические элементы
Footer обязателен на всех страницах и должен содержать:
• Политику конфиденциальности
• Согласие на обработку персональных данных
• Cookie-политику
• Настройки cookie
• Служебные и контактные ссылки по структуре сайта
Cookie-баннер обязателен при первом заходе на сайт. Пользователь должен иметь возможность: принять cookie, отклонить необязательные cookie или открыть настройки cookie.
В настройках cookie должна быть возможность отключать аналитические cookie. Если на сайте подключена Яндекс.Метрика, она должна загружаться только после согласия пользователя.
6. Адаптив и макеты Figma
Основной desktop-макет в Figma делать под 1200 px. Дополнительно: 768 px (tablet) и 360 px (mobile). Desktop 1440 px как обязательный макет не требуется.
Финальное решение по визуалу и компоновке страниц принимается по макету в Figma. Исполнитель присылает ссылку на Figma на согласование до финальной реализации.
Требования к адаптиву:
• без горизонтального скролла;
• карточки и блоки должны перестраиваться;
• mobile должен быть полноценным, а не урезанным;
• на внутренних страницах обязательны breadcrumbs;
7. Стек

8. Контент
Для сайта-новостника контент берется по тематике новостей, статей, обзоров и аналитики по китайским автомобилям и ноускатам.
Для сайта-заявки контент должен быть собран так, чтобы сайт выглядел как рабочий сервис для подбора и заказа ноускатов из Китая.
Картинки подбираются под тематику сайта из бесплатных стоков или из материалов заказчика.
